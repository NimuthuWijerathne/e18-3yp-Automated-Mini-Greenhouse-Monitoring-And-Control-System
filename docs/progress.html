Progress

Progress Plan

This is the initial progress plan we had at the project proposal. we gave a main priority to discussing and finalizing the project idea. Then we allocated time to learnt he software as some technologies were new to us. Since we had a lot of hardware components in our project, we allocated a time to buy all those things we need. Initially our plan was to build the hardware first and then the software. But with the time constraints and the flow, we did the opposite, building the software first and the circuit digarams, and then moving on to the physical product.
